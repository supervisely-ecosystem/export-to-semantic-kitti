This app works only with Point Cloud Episodes projects. Each dataset becomes a separate sequence in SemanticKITTI format.
Point clouds will be saved in binary format .bin (x, y, z, intensity as float32)
Semantic labels will be saved in binary format .label (uint32: lower 16 bits = semantic class ID, upper 16 bits = instance ID)
Export will create SemanticKITTI format structure:
sequences/
  00/
    velodyne/ # point clouds (.bin)
    labels/ # semantic labels (.label)
semantic-kitti.yaml # class mappings
